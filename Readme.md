# Basic C++ Programs for Beginners

This repository contains 5 simple C++ programs designed for students and beginners learning the fundamentals of C++. Each program focuses on a key concept, such as loops, conditionals, and user input, making it perfect for practice and educational purposes.

## Programs Included

1. **Calculate the Average of Grades**
   - A program that calculates the average of three grades and checks if the student has passed or failed.

2. **Check if a Number is Even or Odd**
   - A program to determine if a given integer is even or odd.

3. **Multiplication Table**
   - Generates the multiplication table for a number provided by the user.

4. **Factorial Calculation**
   - Computes the factorial of a positive integer entered by the user.

5. **Guessing Game**
   - A fun program where the user tries to guess a randomly generated number between 1 and 100.
